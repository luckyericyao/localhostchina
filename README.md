# Localhost Global

Travel Back Home.

Localhost Global is a quiet local-host travel network for thoughtful travelers who want to understand places through people. The first launch market is China, with the brand designed to expand beyond one country over time.

## Development

This project uses Next.js, TypeScript, Tailwind CSS, and the App Router.

```bash
pnpm install
pnpm dev
```

## Validation

```bash
pnpm lint
pnpm build
```
